A set of examples using JEP. The following examples are provided:
  • SimpleTest - The simplest example that demonstrates the use of JEP for eval_xuation of a single expression
  • Console - A simple command-line eval_xuator.
  • eval_xuator - An applet that eval_xuates expressions entered.
  • FunctionPlotter - An applet that plots arbitrary functions entered by the user.
  • Fractal - A fractal drawing applet based on an arbitrary function.
  • JEPTester - Tests JEP eval_xuation by comparing expressions from consecutive lines of a text file.
  • ThreadTest - A brute force test for detecting mutlithreading issues
  • LargeExpressionTest - A simple class to test eval_xuation of large expression.
